## 📌 Usage

To use these banners in your GitHub README, copy and paste the following markdown:


```html
<p align="center">
  <a href="https://ishan-oshada.vercel.app/">
    <img src="https://ishan-rest.vercel.app/svg/banner/dev/text" alt="Banner" style="width:100%;">
  </a>
</p>
```
